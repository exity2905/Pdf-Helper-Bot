# Функционал PDF Helper Bot

## Стартовая версия

- Фото документа -> PDF-скан.
- PDF -> сжатый PDF.
- PDF -> отдельные страницы.
- PDF -> текст.
- PDF -> краткое AI-summary.

## Следующий этап

- Несколько фото -> один PDF.
- Объединение нескольких PDF.
- PDF -> Word.
- Word -> PDF.
- OCR для сканов.
- Вопросы по документу.
- Добавить подпись, дату, текст.
- Замазать или скрыть текст.

## Платные функции

- Большие файлы.
- Пакетная обработка.
- OCR.
- AI-summary и вопросы по документу.
- PDF -> Word с сохранением верстки.
