# Функционал PDF Helper Bot

## Стартовая версия

- Выбор языка при старте: украинский, русский, английский.
- Украинский язык по умолчанию.
- Фото документа -> PDF-скан.
- PDF -> сжатый PDF.
- PDF -> отдельные страницы.
- Несколько PDF -> один PDF.
- PDF -> текст.
- PDF -> Word через текстовый слой.
- Word -> PDF через LibreOffice.
- PDF -> краткое AI-summary.

## Следующий этап

- Несколько фото -> один PDF.
- OCR для сканов.
- Вопросы по документу.
- Добавить подпись, дату, текст.
- Замазать или скрыть текст.

## Платные функции

- Большие файлы.
- Пакетная обработка.
- OCR.
- AI-summary и вопросы по документу.
- PDF -> Word с сохранением верстки.
