# PDF Helper Bot

Telegram-бот для работы с документами: фото в скан, PDF-инструменты, OCR, конвертация и AI-помощник по документам.

## Что умеет каркас

- показывает выбор языка при старте: украинский, русский или английский;
- использует украинский язык по умолчанию;
- принимает фото документа и делает аккуратный PDF-скан;
- принимает PDF и показывает кнопки действий;
- сжимает PDF базовым способом;
- разделяет PDF на отдельные страницы;
- достает текст из обычного PDF;
- конвертирует Word -> PDF с предобработкой таблиц;
- конвертирует PDF -> Word через текстовый слой;
- готов к добавлению OCR и AI-вопросов по документу.

## Быстрый запуск

1. Создай бота через `@BotFather`.
2. Скопируй `.env.example` в `.env`.
3. Вставь токен в `BOT_TOKEN`.
4. Установи зависимости:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

5. Запусти:

```bash
python -m app.main
```

## Railway

Для деплоя на Railway добавлены `Dockerfile` и `railway.json`.

Переменные окружения:

```env
BOT_TOKEN=токен_из_BotFather
OPENAI_API_KEY=
MAX_FILE_MB=20
```

Start command:

```bash
python -m app.main
```

`OPENAI_API_KEY` можно оставить пустым, пока AI-summary не подключён.

## Команды

- `/start` - выбор языка и главное меню.
- `/help` - короткая инструкция.

## MVP-функции

1. Фото -> PDF-скан.
2. Несколько фото -> один PDF.
3. Сжатие PDF.
4. Объединение PDF.
5. Разделение PDF.
6. PDF -> Word.
7. Word -> PDF.
8. OCR: скан -> текст.
9. AI-summary документа.
10. Вопросы по документу.

## Важные заметки

- Обычные Telegram-боты скачивают файлы через Bot API с ограничением размера. Для больших файлов лучше поднимать локальный Bot API Server.
- Для OCR на сервере нужен `tesseract`.
- Для Word/PDF-конвертации на сервере часто используют LibreOffice в headless-режиме.
